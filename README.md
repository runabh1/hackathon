MindScape, a beautiful mental wellness journey app! This is a a calming, interactive experience with soft pastels, smooth animations, and wellness tools.

Design Inspiration: Drawing from meditation apps like Headspace and Calm, with ethereal gradients, gentle animations, and a journey-like progression through different peaceful environments.

Features for this version:

Landing page with time-shifting gradient background
Interactive story journey (forest → ocean → space scenes)
Wellness tools (breathing exercise, focus timer, affirmations)
Mood tracker with local storage and canvas visualization
Achievement badges system
tech stack used : html , css and javascript
